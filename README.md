# Number Guessing Game (Python)

This is a beginner-level Python project where the computer randomly selects a number and the user tries to guess it.

## Features
- Random number generation
- User input handling
- Error handling using try-except
- Count number of attempts

## Technologies Used
- Python

## How to Run
1. Install Python
2. Open terminal or command prompt
3. Run the program using:
   python main.py
